# Functionele Header - met Button

In dit voorbeeld tonen we hoe een [vl-button](/docs/components-atom-button--documentatie) component kan toegevoegd worden aan de [vl-functional-header](/docs/components-atom-functional-header--documentatie).

We gebruiken een [vl-button](/docs/components-atom-button--documentatie) component gecombineerd met de [vl-group](/docs/styles-layout-group--documentatie) stijl om een actie knop rechts uit te lijnen naast een breadcrumb of andere variant.

In dit geval wordt custom CSS meegegeven om deze layout mogelijk te maken.
