# Functional Header - Voorbeeld Met Search

In dit voorbeeld tonen we hoe een [vl-search](/docs/components-block-search--documentatie) component kan toegevoegd worden aan de [vl-functional-header](/docs/components-block-functional-header--documentatie).

We gebruiken een [vl-search](/docs/components-block-search--documentatie) component gecombineerd met de [vl-group](/docs/styles-layout-afnemers-group--documentatie) stijl om een zoekveld rechts uit te lijnen naast een breadcrumb of andere variant.

In dit geval wordt custom CSS meegegeven om deze layout mogelijk te maken.
