# Functional Header - Voorbeeld Met Button

In dit voorbeeld tonen we hoe een [vl-button-next](/docs/components-next-button--documentatie) component kan toegevoegd worden aan de [vl-functional-header](/docs/components-functional-header--documentatie).

We gebruiken een [vl-button-next](/docs/components-next-button--documentatie) component gecombineerd met de [vl-group-next](/docs/styles-next-layout-afnemers-group--documentatie) stijl om een actie knop rechts uit te lijnen naast een breadcrumb of andere variant.

In dit geval wordt custom CSS meegegeven om deze layout mogelijk te maken.
