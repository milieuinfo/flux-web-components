## Native elementen

Gebruik voor de volgende elementen native HtmlElements met css classes:
- Form: native HTMLFormElement met class `vl-form` - `<form class="vl-form">`
- Action Group: native HtmlDivElement met class `vl-action-group` - `<div class="vl-action-group">`
- Button: `vl-button` component - [documentatie](/docs/components-atom-button--documentatie)

## Grid

Maak voor het opbouwen van de grid structuur gebruik van de [vl-grid classes](/docs/styles-layout-grid--documentatie).<br/>
Plaats deze op native HtmlDivElements.

## Form data

Gebruik de [parseFormData](/docs/ontwerp-form-form-data--documentatie) helper functie om de form data te verzamelen.<br/>
Indien je meer controle nodig hebt kan je zelf de [FormData](https://developer.mozilla.org/en-US/docs/Web/API/FormData) API gebruiken.
