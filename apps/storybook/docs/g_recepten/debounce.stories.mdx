# Debounce (Common)

De `debounce` methode beperkt het aantal keren dat een functie aangeroepen wordt.

## Voorbeeld

```ts
import { debounce } from '@domg-wc/common';

document.addEventListener('DOMContentLoaded', () => {
    const inputField = document.getElementById('input-field'); // <vl-input-field id="input-field"></vl-input-field>
    if (inputField) {
        inputField.addEventListener(
            'vl-input',
            debounce((event) => {
                // doe iets
            }, 200)
        );
    }
});
```
