# Stacked

<FluxComponentMetaData id="styles-layout-stacked"/>


## Doel

De 'stacked' style-classes beïnvloeden de ruimte tussen kind items binnen Flexbox of Grid containers


## Gebruik

Op de omsluitende parent de class `vl-stacked-large`, `vl-stacked-medium` of `vl-stacked-small` zetten,
 voegt witruimte toe tussen de kinderen. Indien je niet in een Grid of Flexbox container zit, dien je
 `vl-stacked` toe te voegen, deze voorziet een Flexbox in kolom layout.
