# Spacer

<FluxComponentMetaData id="styles-layout-spacer"/>


## Doel

De 'spacer' style-classes voorzien onderaan (extra) witruimte.


## Gebruik

De spacers kunnen op een willekeurig element gebruikt worden. In de voorbeelden hieronder zie je telkens 2
separators met op de eerste de desbetreffende spacer.

### vl-spacer

Voeg onderaan een marge van 2rem toe.

<Canvas of={VlSpacerStories.SpacerDefault} />

### vl-spacer-xxsmall

Voeg onderaan een marge van 0.5rem toe.

<Canvas of={VlSpacerStories.SpacerXXSmall} />

### vl-spacer-xsmall

Voeg onderaan een marge van 1rem toe.

<Canvas of={VlSpacerStories.SpacerXSmall} />

### vl-spacer-small

Voeg onderaan een marge van 1.5rem toe.

<Canvas of={VlSpacerStories.SpacerSmall} />

### vl-spacer-medium

Voeg onderaan een marge van 3rem toe.

<Canvas of={VlSpacerStories.SpacerMedium} />

### vl-spacer-large

Voeg onderaan een marge van 6rem toe.

<Canvas of={VlSpacerStories.SpacerLarge} />

### vl-spacer-none

Verwijder de marge onderaan.

<Canvas of={VlSpacerStories.SpacerNone} />
