## Doel

Vanuit Digitaal Vlaanderen worden er 3 fonts voorzien: `Flanders Art Sans`, `Flanders Art Serif` en
`vlaanderen-icon-classic`.

Het `Flanders Art Sans` font is het default font, wat zowel voor de normale als de italic styling dient.
`Flanders Art Serif` is het accent font. Deze fonts zijn beschikbaar in de gewichten:
300, 400, 500 en 700.

[Alle iconen](/docs/components-next-icon--documentatie#overzicht) komen uit het font `vlaanderen-icon-classic`.
