## Doel

De 'spacer' style-classes voorzien onderaan (extra) witruimte.

## Gebruik

De spacers kunnen op een willekeurig element gebruikt worden. In de voorbeelden hieronder zie je telkens 2
separators met op de eerste de desbetreffende spacer.

### vl-spacer-next

Voeg onderaan een marge van 2rem toe.
