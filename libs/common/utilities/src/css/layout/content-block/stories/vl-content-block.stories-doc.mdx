## Doel

`vl-content-block-next` is een layout component die gebruikt wordt om content te groeperen. Het element krijgt een vaste
 breedte en wordt in het midden gecentreerd. Dit is equivalent van wat bij Digitaal Vlaanderen als
 [Layout](https://overheid.vlaanderen.be/webuniversum/v3/documentation/grid) wordt aangeboden.


## Gebruik

### Default

Gebruik de `.vl-content-block-next` CSS class om een content block te definiëren.

<Canvas of={VlContentBlockStories.ContentBlockDefault} />

### Full width

Gebruik de `.vl-content-block-next--full-width` CSS class om een content block de volledige breedte te laten innemen. Dit is een modifier class die steeds op een element met de `.vl-content-block-next` class moet geplaatst worden.
