# Search Result

<VluxMetaData id="components-search-result" />

Gebruik de `search-result` component om een zoekresultaat weer te geven.<br/>

Volgende sub-componenten zijn beschikbaar om het zoekresultaat op te bouwen:

- VlSearchResultProperties `[vl-search-result-properties]`
- VlSearchResultText `[vl-search-result-text]`
- VlSearchResultTitle `[vl-search-result-title]`


## Voorbeeld

```js
import { VlSearchResultComponent } from '@domg-wc/components';
```

```html
<vl-search-result></vl-search-result>
```
