# Icon

Gebruik de `icon` component om een icoon af te beelden op een pagina.<br/>
Zie het [overzicht](#overzicht) hieronder voor alle beschikbare iconen.


## Voorbeeld

```js
import { VlIconComponent } from '@domg-wc/components/atom';
```

```html
<vl-icon></vl-icon>
```

### Label (toegankelijkheid)

Standaard wordt een icon als decoratief beschouwd en wordt er `aria-hidden` op geplaatst. Indien het relevant is voor de
inhoud en gelezen moet worden door screenreaders moet er een `label` meegegeven worden.
