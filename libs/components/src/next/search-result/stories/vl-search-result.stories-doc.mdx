# Search Result - next

<VluxMetaData id="components-next-search-result" />

Gebruik de `search-result-next` component om een zoekresultaat weer te geven.<br/>

Volgende sub-componenten zijn beschikbaar om het zoekresultaat op te bouwen:

- VlSearchResultProperties `[vl-search-result-properties-next]`
- VlSearchResultText `[vl-search-result-text-next]`
- VlSearchResultTitle `[vl-search-result-title-next]`


## Voorbeeld

```js
import { VlSearchResultComponent } from '@domg-wc/components/next/search-result';
```

```html
<vl-search-result-next></vl-search-result-next>
```
