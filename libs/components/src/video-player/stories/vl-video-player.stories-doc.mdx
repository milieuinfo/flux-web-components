# VideoPlayer

<VluxMetaData id="components-video-player" />

Gebruik de `video-player` component om een video-player af te beelden.<br/>

Deze component is de nieuwe versie van het [vl-video-player](/docs/elements-video-player--documentatie) element, we raden aan om op termijn deze versie te gebruiken.


## Voorbeeld

