# Wizard Pane

<FluxMetaData id="components-block-wizard-wizard-pane" />

Gebruik de `wizard-pane` component om een stap in de wizard af te beelden.
Te gebruiken in combinatie met de [wizard](/docs/components-block-wizard-wizard--documentatie) component


## Voorbeeld

```js
import { VlWizardPane } from '@domg-wc/components/block';
```

```html
<vl-wizard>
    <vl-wizard-pane name="Step 1"><p>Pane content</p></vl-wizard-pane>
</vl-wizard>
```
