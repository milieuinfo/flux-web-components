## Doel

Gebruik de `content-header` component bovenaan informerende pagina's. Hij biedt gebruikers alle informatie over de
website die ze bezoeken.


## Voorbeeld

```js
import { VlContentHeaderComponent } from '@domg-wc/components/block';
```
