# Content Header

<FluxComponentMetaData id="components-block-content-header" />


## Doel

Gebruik de `content-header` component bovenaan informerende pagina's. Hij biedt gebruikers alle informatie over de
website die ze bezoeken.


## Voorbeeld

```js
import { VlContentHeaderComponent } from '@domg-wc/components/block';
```

```html
<vl-content-header></vl-content-header>
```

<Canvas of={VlContentHeaderStories.ContentHeaderDefault} />

<ArgTypes of={VlContentHeaderStories.ContentHeaderDefault} />


## Toegankelijkheid

<FluxAlert type="info">
{`
    Plaats deze component binnen een \`<main>\`, \`<section>\`, \`<article>\` of \`<aside>\`, zodat
    de interne \`<header>\` geen impliciete banner-role krijgt. De banner-role is voorbehouden voor
    de globale \`<vl-header>\` en moet uniek zijn per pagina.
`}
</FluxAlert>
