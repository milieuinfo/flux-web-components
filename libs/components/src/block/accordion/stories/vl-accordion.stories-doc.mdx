# Accordion

<FluxComponentMetaData id="components-block-accordion" />


## Doel

Gebruik de `accordion` component om informatie te tonen of te verbergen aan de hand van een toggle.
Voor meer informatie over het afbeelden van meerdere accordions zie de
[accordion-list](/docs/components-block-accordion-list--documentatie) component.


## Voorbeeld

```js
import { VlAccordionComponent } from '@domg-wc/components/block';
```

```html
<vl-accordion></vl-accordion>
```
