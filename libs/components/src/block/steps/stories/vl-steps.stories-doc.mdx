# Steps

<FluxComponentMetaData id="components-block-steps-steps" />


## Doel

Gebruik de `steps` component om een verticale lijst van stappen af te beelden om de gebruiker door een procedure te
begeleiden.<br/>

Deze component is de nieuwe versie van de [vl-steps](/docs/components-block-steps--documentatie) component, we raden aan deze versie te gebruiken.


## Voorbeeld

```js
import { VlStepsComponent } from '@domg-wc/components/block';
```

```html
<vl-steps></vl-steps>
```
