# Textarea

<FluxComponentMetaData id="components-form-textarea" />


## Doel

Gebruik de `textarea` component om een textarea veld toe te voegen aan een pagina.<br/>
Zie het [form demo](/docs/patronen-formulier-demo--documentatie) voorbeeld voor het gebruik binnen een form.

Deze component bevat geen rich-text modus: gebruik daarvoor
[vl-textarea-rich](/docs/components-form--text-area-rich--documentatie).


## Voorbeeld

```js
import { VlTextareaComponent } from '@domg-wc/components/form';
```

```html

```
