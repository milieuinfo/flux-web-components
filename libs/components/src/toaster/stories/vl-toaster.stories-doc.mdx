# Toaster

<VluxMetaData id="elements-toaster" />

Gebruik een `toaster` om de gebruiker te informeren over een actie of status van een actie.


## Voorbeeld

```js
import { VlToaster } from '@domg-wc/components';
```

```html
<div is="vl-toaster"></div>
```

<Canvas of={VlToasterStories.ToasterDefault} />


## Configuratie

<ArgTypes of={VlToasterStories.ToasterDefault} />

## Gebruik

Definiëer de `vl-toaster` als laatste element in de body van je HTML. Dit zorgt ervoor dat de toaster bovenaan de pagina verschijnt zonder dat de z-index van andere elementen moet worden aangepast.
