# Map Draw Line Action

<FluxComponentMetaData id="map-components-action-draw-line-action" />


## Doel
Gebruik de `map-draw-line-action` component om een lijn te tekenen op een
[map-features-layer](/docs/map-layer-vector-layer-features-layer--map-features-layer-default).<br/>
Deze component erft over van de `VlMapDrawAction` klasse, die op zijn beurt overerft van de `VlMapLayerAction` klasse,
die op zijn beurt overerft van de `VlMapAction` klasse.


## Voorbeeld

```js
import { VlMapDrawLineAction } from '@domg-wc/map';
```

```html
<vl-map-draw-line-action></vl-map-draw-line-action>
```
