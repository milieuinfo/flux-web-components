# Select Location

<FluxMetaData id="map-select-location" />

Gebruik de `vl-select-location` component om een zoekbalk af te beelden waarmee er een adres geselecteerd kan worden.

Deze component erft over van de [vl-select-rich](/docs/components-form-select-rich--documentatie) component.


## Voorbeeld

```js
import { VlSelectLocationComponent } from '@domg-wc/map';
```

```html
<vl-select-location></vl-select-location>
```
