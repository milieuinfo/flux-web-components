# Map Side Sheet

<FluxComponentMetaData id="map-components-side-sheet" />


## Doel

Gebruik de `map-side-sheet` component om een uitklapbaar zijpaneel af te beelden over de map.<br/>
Deze component erft over van de [side-sheet](/docs/components-block-side-sheet--side-sheet-default) component en heeft ook al diens functionaliteit.<br/>
**Let op**: de default positie van de `map-side-sheet` is links, terwijl de default positie van de
[side-sheet](/docs/components-block-side-sheet--side-sheet-default) rechts is.


## Voorbeeld

```js
import { VlMapSideSheet } from '@domg-wc/map';
```

```html
<vl-map-side-sheet></vl-map-side-sheet>
```
